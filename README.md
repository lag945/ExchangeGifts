# ExchangeGifts
Exchange gifts and NO bugs :)

# [Online Demo](https://lag945.github.io/ExchangeGifts/index.htm)

# 原理
將參加人數n人，每個人不重複至編號1~n，程式會隨機抽出每個配對，如3號抽獎者抽到5號抽獎者的禮物等...
此程式每個人都會抽到，且不會抽到自己的禮物。
